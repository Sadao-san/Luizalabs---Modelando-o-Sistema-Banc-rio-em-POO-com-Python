# Sistema Bancário em POO com Python  
Bootcamp DIO + Luizalabs — Desafio 2

## Descrição
Projeto desenvolvido para o desafio "Modelando o Sistema Bancário em POO com Python", com foco na aplicação de Programação Orientada a Objetos.

O sistema permite criar clientes, contas e realizar operações bancárias básicas.

## Funcionalidades
- Criar cliente
- Criar conta corrente
- Depositar valores
- Sacar valores com limite
- Exibir extrato
- Listar contas

## Estrutura
```classes/```
```services/```
```ui/```
```main.py```

## Conceitos Utilizados
- Programação Orientada a Objetos
- Classes abstratas (ABC)
- Encapsulamento e herança
- Separação de responsabilidades

## Como Executar
```git clone https://github.com/seu-usuario/seu-repositorio.git```

```cd seu-repositorio```
```python main.py```
